<h1>Elastic SIEM Home Lab</h1>
This project is a home security information and event management (SIEM) lab using Elastic Stack and a Kali VM to simulate threat activity and generate alerts.

## Goals / Features:
- Setting up Elastic SIEM.
- Installing agent on Kali VM.
- Generating Nmap scans and detecting them.
- Building dashboards for log analysis.
- Creating alert rules for scanning.
- Setting up SMTP-based alerting.
